FPDF2 Make Font

The extension must be either .ttf, .otf or .pfb and determines the font type.
The encoding defines the association between a code (from 0 to 255) and a character.
Indicates whether the font should be embedded in the PDF or not.
Indicates whether sub-setting should be used, that is to say, whether only the characters from the selected encoding should be kept in the embedded font.
